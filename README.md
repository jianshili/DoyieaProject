# 翼动体育

  应用介绍:"翼动体育"app是一款线上预约排球场馆和购买排球用品的应用。

  应用功能:

    1、首页：特惠专区、新店开业、明星单品、新闻资讯、推荐场馆、单品推荐等类型。
    
    2、分类：排球馆分类按照室内排球馆、室外排球馆等两大类；
        排球场馆：预约某场馆后需在预约的时间内到达场馆进行使用。

    3、商城：排球、护具等类型商品；
        下单后需用户持取货码到店取货，并且在店内付款。

    4、订单：商品订单、场馆预约等类型订单。
  
    5、我的：个人信息、商品收藏数量、优惠券数量、场馆收藏数量和相关设置等。
